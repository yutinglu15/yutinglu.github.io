Basic Knowledges

  1. What is overfitting/underfitting?

    Overfitting means that the result fits too close to the training data. Thus it might approach great performance for training data, but perform poorly on unseen data.

    Avoid overfitting

    1. Cross validation

    2. Obtain more training data

    3. Data augmentation (Obtain more training data)

    4. Regularization (regularize the loss)

    5. Feature selection

    6. Early stop

    7. Ensemble model

    Underfitting is the model is too simple to capture the relationship of training data. The performance will not be good for both training data and testing data. It occurs when the model is too simple or the data is not enough, training time not enough.

  2. What is bias/variance trade off?

    Bias means the difference between prediction of the model and the actual correct value.

    High bias Variance means the variability of the prediction of the model. It indicates the spread of the data.

  3. Difference between Generative and Discriminative?

    Generative model: Learn the model based on joint probability p(x, y). Can be used for data generation.

    Discriminative model: Learn the model based on conditional probability p(x|y). Often used for classification.

  4. Give a set of ground truths and 2 models, how do you be confident that one model is better than another?

    Train and test on same dataset. Set fix metrics to evaluate.

Reguarlization

  1. 1. L1 vs L2, which one is which and difference?

    L1 norm: lambda of summation of absolute value of the coefficients. used for sparse model/feature selection.

    L2 norm: lambda of summation of squared of coefficients.

  2. Lasso/Ridge?

    Ridge adds an L2 norm penalty. It is Laplace prior.

    Lasso adds an L1 norm penalty. It is Gaussian prior.

  3. Why L1 norm is sparser than L2 norm?

    That L1 norm is easier to get extreme value at x=0 compared to L2 norm.

  4. Why regularization reduced overfitting?

    It applies a penalty for the coefficient to let them approach zero. It decrease the complexity of the model.

  5. Why regularization use L1 L2 instead of L3, L4?

    L1 L2 cover most of the application scenario. L3 L4 can be a linear combination of L1 and L2.

Metrics

  1. What is precision and recall, what is the trade-off?

    Precision = True Positive/(True Positive + False Positive)

    It is used when the cost of false positive is higher. False Positive means a false prediction of presence of something when it actually didn’t happened.

    Recall = True Positive/(True Positive + False Negative)

    It is used when the cost of false negative is higher. False negative means a false prediction of something not present while in fact it does present.

    If the precision increase then the recall will decrease. If the recall increase then the precision will decrease as well.

  2. What metric to use when data unbalanced?

    F1 score: it is a weighted average of precision and recall.

  3. What metric to use for classification problem?

    (not sure)

  4. What is confusion matrix?

    It evaluate the quality of the output of a classifier by putting the result of predicted label against actual label.

  5. What is ROC and AUC?

    ROC curve is a graph indicate the performance of a classification model. The x-axis is false positive rate, and the y-axis is the true positive rate. AUC means the area under the ROC curve. It can be interpreted as ==the probability that the model ranks a random positive example more highly than a random negative example==.

    False positive rate = False Positive/(False Positive + True Negative)

    True positive rate = True Positive/(True Positive + False Negative)

    A model whose predictions are 100% wrong has an AUC of 0.0; one whose predictions are 100% correct has an AUC of 1.0.

    AUC is:

    scale-invariant - It measures how good the prediction is rather than the absolute correctness.

    classification-threshold-invariant - It measures the quality of the prediction irrespective to what threshold is chosen.

  6. What is Log-loss?

    It is negative average of log of predict probabilities for each correctly predict instance.

    Used when the classifier are too confident about an incorrect classification.

Loss and Optimization

  1. Why MSE cannot work for logistic regression?

    Because MSE (mean square error) loss function is not convex. It is hard to find global minimum. Instead logistic regression use MLE (maximum likelihood estimation).

  2. Formula of MSE, when is it use?

    It is always used in Linear regression.
